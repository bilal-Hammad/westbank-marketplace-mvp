# Flutter Stub
هذا مجلد قالب (Stub) لتطبيق Flutter — ليس مشروع Flutter مولّد بالكامل.
لإنشاء مشروع Flutter:
1) flutter create mobile_app
2) انسخ ملفات lib من هذا المجلد إلى مشروعك
3) أضف dependencies المذكورة داخل الملفات

السبب: بيئة التشغيل هنا لا تولّد Flutter project بالكامل، لكن الكود جاهز للنسخ.
